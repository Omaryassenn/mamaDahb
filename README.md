# Mama Dahb

# Description
A landing page for mama dahb resturant

# Getting Started
## Run Locally

Clone the project

```bash
  git clone https://github.com/Omaryassenn/mamaDahb
```

Go to the project directory

```bash
  cd Fetching_info
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
